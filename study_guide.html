세윤이의 코딩 공부방 🌟

우리가 함께 만든 게임 월드를 공부하고 직접 실행해 보세요!

👋 반가워요, 미래의 게임 제작자님!

우리는 방금 인터넷에서 게임 코드를 다운로드하고, 홈 화면의 이름을 "세윤이의 게임 월드"로 바꾼 뒤 다시 인터넷 저장소(GitHub)로 안전하게 올려보냈어요. 이 모든 과정에서 쓰인 핵심 도구들과 방법을 알아봅시다!

🧩 중요한 코딩 삼총사

💡 1. 스트림릿 (Streamlit)

파이썬 코드를 마법처럼 예쁜 웹사이트로 만들어 주는 도구예요. 디자인 코드를 직접 쓰지 않아도 알아서 버튼과 텍스트를 이쁘게 그려줘요!

⏰ 2. 깃 (Git)

코드의 타임머신이에요. 코드가 작동하지 않거나 실수했을 때, 옛날에 잘 저장해 둔 시점으로 언제든지 돌아갈 수 있게 기록을 남겨 줘요.

☁️ 3. 깃허브 (GitHub)

내 코드가 지워지지 않도록 인터넷 클라우드에 안전하게 보관하고, 다른 친구들에게 내 게임을 보여주는 온라인 전시장이에요.

📁 우리 폴더의 보물 지도

현재 폴더 안에는 다음과 같은 소중한 코드 파일들이 들어있어요:

📄 streamlit_app.py
우리 웹사이트의 대문(홈페이지) 역할을 하는 메인 파일이에요. 타이틀 글씨를 수정한 곳이에요!
🎮 up_down_game.py
pages 폴더 안에 들어있어요. 컴퓨터가 생각한 숫자를 맞추는 업다운 게임 소스코드예요.
🗒️ requirements.txt
이 프로그램을 실행하기 위해 설치해야 하는 파이썬 패키지(도구)의 목록 적힌 영수증이에요.

🚀 내 컴퓨터에서 게임 실행하기

수정한 게임 월드를 내 브라우저에서 켜보고 싶다면 터미널에 아래 명령어를 치면 돼요:

python -m streamlit run streamlit_app.py

💡 이 명령어를 치고 나면 브라우저 창이 열리면서 http://localhost:8501 주소에서 내 게임이 실행되는 것을 볼 수 있어요!

☁️ 깃허브(GitHub)에 내 코드 동기화하기

우리가 코드를 수정하고, 인터넷 깃허브로 보낼 때 사용한 4단계 흐름이에요:

  • 1
    파일 준비하기 (Stage)
    내가 수정한 파일들을 택배 상자에 담는 과정이에요.
    git add streamlit_app.py
  • 2
    포장하고 이름표 붙이기 (Commit)
    상자를 포장하고 어떤 걸 고쳤는지 메모지(이름표)를 붙여요.
    git commit -m "Update title: 세윤이의 게임 월드"
  • 3
    인터넷으로 트럭 발송하기 (Push)
    포장한 택배 상자를 인터넷 깃허브 저장소로 쏘아 올려요.
    git push -u origin main
